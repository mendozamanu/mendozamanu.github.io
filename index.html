MAPLID: a new multi-label approach for place identification using data supplied by mobile network operators

MAPLID introduces a supervised multi-label approach for place identification using data supplied by mobile network operators. The framework evaluates several multi-label classifiers and validates results in different geographical settings, incorporating multimodal Point of Interest (POI) data to enhance robustness.
