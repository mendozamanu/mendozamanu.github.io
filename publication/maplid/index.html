MAPLID: a new multi-label approach for place identification using data supplied by mobile network operators

Manuel Mendoza-Hurtado, Juan A. Romero-del-Castillo, Domingo Ortiz-Boyer
2026
DOI

Abstract

This paper introduces MAPLID, a supervised multi-label approach for place identification using data supplied by mobile network operators. The framework evaluates several multi-label classifiers and validates results in different geographical settings, incorporating multimodal Point of Interest (POI) data to enhance robustness. MAPLID addresses the challenge of identifying land use types at a fine spatial granularity using mobile network data, demonstrating the advantages of a multi-label formulation over traditional single-label approaches.
